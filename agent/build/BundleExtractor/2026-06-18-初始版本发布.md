# 2026-06-18 - 初始版本发布

## 概述

BundleExtractor 是为支持 Steam/Epic 版本 PoE2 而开发的命令行提取工具，用于从 `.bundle.bin` 文件中提取指定文件。

## 功能说明

### 命令行用法

```powershell
BundleExtractor.exe <index.bin> <file_path> <output_path>
```

- `index.bin`：游戏目录下的 `Bundles2\_.index.bin` 路径
- `file_path`：bundle 内的文件路径（如 `data/balance/baseitemtypes.datc64`）
- `output_path`：输出文件保存路径

### 示例

```powershell
# 提取英文版 BaseItemTypes
BundleExtractor.exe "C:\Path of Exile 2\Bundles2\_.index.bin" "data/balance/baseitemtypes.datc64" "output_en.datc64"

# 提取繁体中文版 BaseItemTypes
BundleExtractor.exe "C:\Path of Exile 2\Bundles2\_.index.bin" "data/balance/traditional chinese/baseitemtypes.datc64" "output_tc.datc64"
```

## 技术实现

- **语言**：C# (.NET 8)
- **核心库**：LibBundle3.dll（社区开源库）
- **依赖**：oo2core.dll（Oodle 压缩库）
- **发布方式**：单文件自包含 exe
