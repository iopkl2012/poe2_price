# 跳过基础通货自身价格标注

## 变更时间
2026-06-19

## 变更内容
- `build_poe2scout_price_patch.py` 增加基础通货判断，识别 `divine` 和 `exalted` API id。
- 在统一生成 `display_price` 时，将 Divine Orb 和 Exalted Orb 自身的显示价格置空。
- 匹配生成补丁行时跳过没有 `display_price` 的观察项，避免生成 `Divine Orb = 1.00D` 或 `Exalted Orb = 1.00E`。
- 同步更新 `build/payload/build_poe2scout_price_patch.py`。

## 变更原因
Divine Orb 和 Exalted Orb 本身就是价格单位，继续标注自身价格会显得冗余且占用物品名空间。
