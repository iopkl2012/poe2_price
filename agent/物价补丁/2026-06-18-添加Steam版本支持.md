# 2026-06-18 - 添加 Steam 版本支持

## 更改概述

为物价补丁工具添加了对 Steam/Epic 版本（Bundles2 格式）的完整支持，实现了自动检测游戏版本并执行对应操作的功能。

## 更改详情

### 1. poe2_patch_common.ps1

- **修改** `Get-Poe2GameMode`：检测逻辑从 `Bundles2\Streaming\Content` 目录改为检测 `Bundles2\_.index.bin` 文件存在性
- **修改** `Get-Bundles2BaseItemsPath` → `Get-Bundles2Paths`：返回 bundle 路径信息（IndexBin、EnBaseItems、TcBaseItems 路径）

### 2. update_price_patch.ps1

- **新增** Steam 模式检测：检查 `BundleExtractor.exe` 和 `oo2core.dll` 存在性
- **新增** `BundleExtractor.exe` 提取逻辑：从 Steam bundle 中提取 `baseitemtypes.datc64`
- **新增** `PatchBundle3.exe` 安装逻辑：创建临时 ZIP 并使用命令行补丁工具安装

### 3. restore_price_patch.ps1

- **新增** Steam 模式检测：检查必要的工具文件
- **新增** `Ensure-CleanBaseItemForRestore` 函数：从 bundle 重新提取干净的 BaseItemTypes
- **修改** 还原逻辑：使用 `PatchBundle3.exe` 替代直接文件复制

### 4. 新增 BundleExtractor 项目

- 创建 `BundleExtractor/` C# 项目
- 使用 `LibBundle3.dll` 从 Steam bundle 中提取文件
- 编译为独立 exe 部署到 `物价补丁/tools/`

## 影响范围

- `物价补丁/tools/update_price_patch.ps1`
- `物价补丁/tools/restore_price_patch.ps1`
- `物价补丁/tools/poe2_patch_common.ps1`
- 新增 `BundleExtractor/` 项目
- 新增 `物价补丁/tools/BundleExtractor.exe`
- 新增 `物价补丁/tools/oo2core.dll`
