# 第三方工具说明

本源码包包含用于 POE2 资源读写的第三方工具。

- `物价补丁/tools/GGPKExtractor`
  - 从国际服官方独立端 `Content.ggpk` 提取物品表。
- `物价补丁/tools/BundleExtractor`
  - 从 `Bundles2\_.index.bin` 指向的 bundle 中提取指定虚拟文件。
- `物价补丁/一键安装特殊补丁工具/PatchBundledGGPK3.*`
  - 把补丁 zip 写入 `Content.ggpk`。
- `物价补丁/一键安装特殊补丁工具/PatchBundle3.*`
  - 把补丁 zip 写入 Bundles2。

本项目只写入当前客户端对应的 `BaseItemTypes.datc64` 虚拟资源。修改游戏文件存在封号或校验风险，请关闭游戏后使用并自行承担风险。
